# Pakistan Open Data (opendata-com-pk)

Pakistan Open Data is a national government open-data portal for Pakistan running **CKAN** (~1,471 datasets).

**URL:** [https://raw.githubusercontent.com/api-evangelist/opendata-com-pk/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/opendata-com-pk/refs/heads/main/apis.yml)

## API
- Base: `https://opendata.com.pk/api/3/action` · [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://opendata.com.pk)
